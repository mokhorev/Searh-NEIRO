# Searh-NEIRO UI

Локальный веб-интерфейс для ежедневной работы без API-ключей.

## Зачем сделан UI

CSV-режим работает, но для ежедневной работы неудобен. Новый интерфейс устроен как рабочая доска:

- слева навигация и прогресс;
- отдельно компании;
- отдельно промпты;
- отдельно список нейросетей;
- отдельная очередь задач;
- отдельный вид по каждой компании;
- встроенная статистика очереди, ошибок и запусков;
- отдельная вкладка отчёта.

## Установка обновления

```powershell
cd C:\Projects\Searh-NEIRO
.\.venv\Scripts\Activate.ps1
git pull
pip install -e .
```

## Запуск интерфейса

```powershell
neirosearch-ui
```

Откроется локальная страница в браузере.

Если команда не сработала:

```powershell
python -m streamlit run src\neirosearch\ui_app.py
```

## Как работать

### 1. Компании

Открой раздел **Компании**.

Добавь строки:

```text
Компания | Ниша / услуга | Город / регион | Конкуренты через запятую
```

Нажми **Сохранить компании**.

### 2. Промпты

Открой раздел **Промпты**.

Вставь список промптов, один промпт на строку.

Доступны переменные:

- `{brand}` — компания;
- `{industry}` — ниша/услуга;
- `{region}` — регион.

Пример:

```text
Кого посоветуешь для услуги: {industry} в городе {region}? Дай 5 вариантов.
Стоит ли выбирать {brand} для {industry} в {region}? Какие есть альтернативы?
```

### 3. Нейросети

Открой раздел **Нейросети**.

Выбери только те, которые реально доступны через браузер/VPN.

### 4. Очередь

Открой раздел **Очередь**.

Нажми **Сгенерировать новую очередь**.

Дальше по каждой задаче:

1. открой нужную нейросеть кнопкой;
2. скопируй промпт;
3. вставь его в нейросеть;
4. скопируй ответ;
5. вставь ответ в поле **Вставь ответ нейросети**;
6. нажми **Сохранить ответ**.

### 5. Вид по каждой компании

Открой раздел **По компаниям**.

Там видно прогресс по каждой компании и каждой нейросети.

### 6. Отчёт

Открой раздел **Отчёт**.

Выбери компанию и нажми **Сформировать файлы отчёта**.

Файлы появятся здесь:

```text
outputs\ui_report\<company>\report.md
outputs\ui_report\<company>\summary.csv
outputs\ui_report\<company>\results.jsonl
```

## Важно

Интерфейс не обходит авторизацию и не ломает веб-сервисы. Он помогает вести очередь, копировать промпты, вставлять ответы и видеть результат по компаниям.
