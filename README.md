# Searh-NEIRO

Инструмент для проверки, какие нейросети рекомендуют компанию, бренд, товар или услугу.

Проект собирает ответы из разных LLM-провайдеров и помогает делать аудит AI-видимости: кто упоминается, кто конкурирует, какой ответ дают нейросети, есть ли ссылки/цитаты и где бренд отсутствует.
